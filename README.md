## Shortcuts

- `Cmd` + `Shift` + `B` - build
- `Cmd` + `T` - go to defintion (you need to build tags first - `make tag`)
- `Shift` + `^` + `\`` - open terminal
- You can use `Cmd` + `Click` in terminal to open files
